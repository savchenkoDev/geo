# Geo Microservice

Микросервис Geo для курса Ruby Microservices.

# Зависимости

- Ruby `2.6.6`
- Bundler `2.2.13`
- Sinatra `2.1.0`
- Geocoder `1.6.6`
- Puma `5.2.2`
# Установка и запуск приложения

1. Склонируйте репозиторий:

```
git@github.com:savchenkoDev/geo.git && cd geo
```

2. Установите зависимости:

```
bundle install
```

3. Запустите приложение:

```
bin/puma
```

# Запуск тестов

```
bin/rspec
```
